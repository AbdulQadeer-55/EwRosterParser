Eurowings Roster Parser (Java Tool)
===================================

How to Use:
1. Place your PDF roster file in this folder.
2. Rename your PDF file to "dutyplan.pdf".
3. Double-click "EwRosterParser.jar" (or run it from the command line).
4. A new file named "roster.ics" will be created in this folder.
5. Import "roster.ics" into your Google Calendar or iOS Calendar.

Configuration:
- Dienste.txt: You can edit this file to change how events are named (e.g., change "C/I" to "Check-In").
- IATA.csv: You can add new airports and their timezones here if you fly to new destinations.

Requirements:
- Java installed on your computer.

Enjoy!
